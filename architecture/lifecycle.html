Step 3
Run
each role independently · swati run
1
Fetch manifest from 0G · verify SHA-256(source) == manifest.sourceHash · load choreography
2
Load identity (--identity-file or SWATI_PRIVKEY_HEX)

4
Attestation phase: all roles broadcast signed attest with choreoId hash ✓ All match → flow starts   ✗ Any mismatch → CHOREO_MISMATCH abort
5
Flow execution (EPP): each Conductor walks the same flow(), executes only its own steps After each act: AppendLog.append() → signed, hash-chained entry
6
Run complete: snapshotLog() → BlobVault.putLogSnapshot() → 0g://0x9b46… · swati registry anchor-log → on-chain proof
