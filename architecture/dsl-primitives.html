DSL Primitives
Primitive Who executes Others Notes
Core messaging
roles.X.do() role, prompt → unknown Role XLLM call with prompt Skip — returns undefined Only role X's Conductor sends to LLM
send(v, A, B) value, from, to Role ASign + emit via transport
Role BBlock until recv, verify sig 		Skip — line is invisible Guaranteed delivery. Typed. Every send has a matching recv.
Branching
choose(R, opts) role, options, evidence Role RLLM call → broadcast result to ALL Await broadcast · branch identically on received value No divergence possible. All roles see the same value.
chooseIf(R, bool) role, boolean Role RBroadcast boolean to ALL Await broadcast · branch identically Boolean branch without LLM call
locally(R, fn) role, fn → Located<T> Role Rfn() → Located<T> with value Located<T> with undefined — opaque Role-private computation. Pass to send() to transfer.
Actions
gate(R, p, fn) role, provider, fn Role RCall Gate Provider · await proof · log proof · broadcast Skip / wait for broadcast result.ok only true if proof issued. Irreversible.
computeSend(A, B, fn) from, to, fn Role Afn() + send in one step Role B: recv Compute and send atomically
Composition & state
invoke(sub, input) choreography, input All roles run sub-choreography Roles not in sub simply wait sub must be a subset of current roles
persist(k, v) key, value All — write shared state — Survives across runs via storage adapter
recall(k) key → value All — read shared state — Reads what persist() wrote
recurse(input) input All — restart flow with new input — Loop without stack overflow
Adapter swap — same choreography, any stack

resolver: StaticResolver — hard-coded pubkeys
storage: LocalFileStorage — local disk
llm: MockLLM — scripted responses
gates: LocalGate — local mock
Production
transport: WireChannel — AXL mesh, binary wire
resolver: EnsResolver / OnChainRegistryResolver
storage: BlobVault — 0G decentralised
llm: AnthropicLLM — Claude Sonnet
gates: KeeperHubGate · UniswapGate
